🤖 Multi-Platform Automation

AutoPost-Lunga

Intelligent automation for publishing content on social networks via official API

📱 TikTok

API: Content Posting API

Scopes: user.info.basic, video.upload

Sending videos as drafts for manual review before publishing. Integration with n8n for complete automation.

▶️ YouTube

API: YouTube Data API v3

Scope: youtube.upload

Automatic video upload with title, description and privacy settings (public/unlisted/private).

📌 Pinterest

API: Pinterest API v5

Scopes: pins:read, pins:write

Automated Pin creation with images or videos, including custom link and description.

🔧 How it works

AutoPost-Lunga uses n8n workflows (no-code automation) to monitor folders, webhooks or schedules. When new content is detected, the system calls the respective official APIs and performs the automatic upload.

Complete flow: OAuth Authentication → File Upload → API Response → Content available on the platform (draft or published according to configuration).
