🤖 Multi-Platform Automation

AutoPost-Lunga

Automação inteligente para publicação de conteúdo em redes sociais via API oficial

📱 TikTok

API: Content Posting API

Escopos: user.info.basic, video.upload

Envio de vídeos como rascunhos (drafts) para revisão manual antes da publicação. Integração com n8n para automação completa.

▶️ YouTube

API: YouTube Data API v3

Escopo: youtube.upload

Upload automático de vídeos com título, descrição e configurações de privacidade (público/não listado/privado).

📌 Pinterest

API: Pinterest API v5

Escopos: pins:read, pins:write

Criação automatizada de Pins com imagens ou vídeos, incluindo link e descrição personalizada.

🔧 Como funciona

O AutoPost-Lunga utiliza workflows no n8n (automação no-code) para monitorar pastas, webhooks ou agendamentos. Quando um novo conteúdo é detectado, o sistema chama as respectivas APIs oficiais e realiza o upload automático.

Fluxo completo: Autenticação OAuth → Upload do arquivo → Resposta da API → Conteúdo disponível na plataforma (rascunho ou publicado conforme configuração).
